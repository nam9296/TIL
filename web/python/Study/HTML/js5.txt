객체를 딕셔너리 형태로 표현하므로 순서가 없다 
 중괄호를 사용함


객체 생성 객체명 : coworkers
<script>
    var coworkers={
    "programmer":"kim",
    "designer":"lee"
}
</script>

"programmer":속성(딕셔너리의 key와 같음)
"kim" : 속성값(딕셔너리의 value와 같음)


객체에 속성을 추가할 때  2가지 방법

document.write("개발자명:"+coworkers.programmer+"<br>");
document.write("디자이너:"+coworkers.designer+"<br>");
